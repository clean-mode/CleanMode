# CleanMode

A focused tool for cleaning your laptop without shutting it down.

CleanMode locks your keyboard, trackpad, and cursor for a moment so you can wipe them down without typing random text into your editor or accidentally clicking something you didn't mean to. Available for macOS, Windows, and Linux.

- **Website:** https://cleanmode.app
- **Support and FAQ:** https://cleanmode.app/support

## About this repository

This is the public-facing repository for CleanMode users. The application source code lives in a separate private repository.

Use this repo to:

- **Download a release** — see [Releases](https://github.com/clean-mode/CleanMode/releases) for installers for macOS (`.dmg`), Windows (`.exe`), and Linux (`.deb` / `.AppImage`).
- **Report a bug** — [open a new issue](https://github.com/clean-mode/CleanMode/issues/new).
- **Request a feature** — same link.
- **Ask a question** — same link.

## Before opening an issue

Many common questions are already answered on the [support page](https://cleanmode.app/support) — install issues, OS permissions, fn-lock behavior, the auto-exit feature, and more.

If your question isn't covered there, please include:

- Your operating system and version (e.g., macOS 14.5, Windows 11 23H2, Ubuntu 24.04)
- The CleanMode version (shown in the About dialog)
- What you expected to happen
- What actually happened
- Steps to reproduce, if applicable

Screenshots help a lot.

## Contact

For private matters (security, billing, account questions), email support@cleanmode.app instead of opening a public issue.
